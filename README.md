## 📝 Getting Started

To run this portfolio locally:

```bash
git clone https://github.com/MandalTuhin/tuhin-mandal-portfolio.git
cd tuhin-mandal-portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.
